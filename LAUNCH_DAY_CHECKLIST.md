## Emergency Procedures

### Critical Bug Found

1. **Assess Severity**:
   - Game crashes? → Critical
   - Save corruption? → Critical
   - Progression blocked? → Critical
   - Visual glitch? → Not critical

2. **If Critical**:
   - Announce issue immediately
   - Post workaround if possible
   - Create hotfix ASAP
   - Test hotfix thoroughly
   - Deploy hotfix
   - Announce fix deployed
   - Apologize to affected players

3. **If Not Critical**:
   - Log in bug tracker
   - Plan fix for next patch
   - Inform community (transparency)

### Platform Issues

If Steam/Itch.io/etc. has issues:
- [ ] Check platform status page
- [ ] Announce delay (not your fault)
- [ ] Keep community informed
- [ ] Wait for platform to resolve
- [ ] Re-attempt launch when ready

### Negative Reviews

If receiving many negative reviews:
- [ ] Read reviews carefully
- [ ] Identify common complaints
- [ ] Respond professionally (if appropriate)
- [ ] Plan fixes for valid issues
- [ ] Communicate fix plans to community

### Marketing Issues

If press/influencers have issues:
- [ ] Respond to inquiries quickly
- [ ] Provide press keys promptly
- [ ] Offer support for technical issues
- [ ] Thank them for coverage
